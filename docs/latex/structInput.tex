\doxysubsection{Detailed Description}
\mbox{\hyperlink{structInput}{Input}} struct for important parameters and filenames where the exciton-\/basis data is to be read in from. 

This struct mainly holds a list of filenames which have been output by the fortran code, listing where all the data is for the eigenstates of our Hamiltonian (the exciton basis data). Also holds a few parameters that aren\textquotesingle{}t specific to any particular pigment or exciton.
